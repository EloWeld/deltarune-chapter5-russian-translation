# Унификация глоссария — статус

## Закрыто

Все расхождения между чанками разных агентов сведены к единому канону
(`unify.py`, плюс патчи цветных NPC). Принятые варианты:

| Оригинал | Канон |
|---|---|
| Asgore | Азгор (**Gorey → Горей** — отдельная строка, не путать) |
| Raly | Рэлли (несклоняемое) |
| Floradinn | Флорадинн |
| Bratty | Брэтти |
| Scythemare | Косшмар / КОСШМАР |
| Snowgrave | Странный путь |
| BootOil | СапожМасло |
| PerpBook | КнигаУлик |
| FloweryScarf | ФлауэриШарф |
| GreenApron | ЗелёнФартук |
| YellowShred | ЖёлтЛоскут |
| RedSplatter | КрасноеПятно |
| BlueString | СиняяНить |
| TrainPlans | ПланПоезда |
| SethSpecs | ОчкиСета |

**Статы** — латиница `HP` / `AT` / `DF` / `LV` / `EXP`, только `TP → ТП`.

**Цветные NPC** — полные формы, род по оригиналу:
`Yellow → Жёлтый`, `Blue → Синий`, `Green → Зелёный`, `Purple → Фиолетовый` (м.р.);
`Orange → Оранжевая`, `Pink → Розовая`, `Aqua → Бирюзовая` (ж.р.).
Склоняются по падежам, род глаголов вокруг имени согласован.

## Решено оставить как есть

- ~40 строк в `chunk_029` (`CuteBoys`, `Sports`, `Family`, `Turtleneck`…) — латиницей.
- `CatDefend`, `HasAntenna` (127–135) — латиницей, движковые флаги.
- Имена монстров ch5 в 145–146 — латиницей.
- `Damage#w/ ICE` (`chunk_036`) — единственное срабатывание валидатора, ложное:
  слэш здесь часть английского «w/», а не управляющий код.

## Известные потери

- `chunk_083` — каламбуры-омофоны дейт-сима переданы по смыслу, игра слов не сохранилась.
- `chunk_081`, ключи `gml_362_0`–`gml_364_0` — реплики без кода спикера,
  род выбран по контексту (предположительно Ноэлль). Стоит глянуть в игре.

## Инструменты

- `validate_all.py` — сводная валидация: покрытие ключей + управляющие коды.
  Считает коды с фиксированной длиной, поэтому не даёт ложных срабатываний на
  `\cYYellow` (чем грешил старый `check_codes.py` из scratchpad).
- `unify.py` — унификация глоссария. Без аргументов dry-run, с `--write` правит `out/`.
