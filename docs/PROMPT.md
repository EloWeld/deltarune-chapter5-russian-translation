# ЖЕЛЕЗНОЕ ПРАВИЛО: управляющие коды не трогать

В строках есть служебные коды движка GameMaker. Их НЕЛЬЗЯ переводить, удалять, добавлять, переставлять или менять. Переноси их в перевод ДОСЛОВНО, на тех же местах относительно текста.

- `^` + цифра (`^1`…`^9`) — пауза. Оставляй как есть.
- `\M0` `\M1` `\M2` … — смена ракурса/говорящего.
- `\E0` `\E1` … — эмоция лица.
- `\T` `\F` + буква/цифра (`\TN`, `\Fa`, `\FN`) — смена спрайта/лица.
- `\C0`…`\C9` — цвет текста.
- `&` — перенос строки внутри реплики. Количество `&` в переводе должно совпадать с оригиналом.
- `%` и `%%` — конец реплики/страницы. Всегда сохраняй в конце.
- `\\` и прочие последовательности с обратным слэшом — копируй символ в символ.
- Любой другой код вида `\` + буква — копируй как есть.

Если строка состоит только из кодов и пробелов (например `\M0 %`) — верни её без изменений.
